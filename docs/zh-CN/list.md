# 历史记录列表

```
atuin history list
```

| 参数           | 描述                                                  |
| -------------- | ----------------------------------------------------- |
| `--cwd/-c`     | 要列出历史记录的目录（默认：所有目录）                |
| `--session/-s` | 只对当前会话启用列表历史（默认：false）               |
| `--human`      | 对时间戳和持续时间使用人类可读的格式（默认：false）。 |
