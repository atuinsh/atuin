   - Add `render_error_block(frame, message, area, ctx)` function
   - Use `Meaning::Alert` style for error icon ("!")
   - Use `Meaning::Base` style for error text
   - Format as "! {error_message}" with word wrap (like text blocks)
   - Add `BlockKind::Error` case to `render_single_block()`
   - Add `BlockKind::Error` case to `calculate_block_height()` (same as Text)

4. In `app.rs`:
   - Modify `generation_error()` to add Error block to blocks vec:
     ```rust
     pub fn generation_error(&mut self, error: String) {
         // Remove spinner
         if let Some(pos) = self.blocks.iter().rposition(|b| b.kind == BlockKind::Spinner) {
             self.blocks.remove(pos);
         }
         // Add error block
         self.blocks.push(Block::new_error(error.clone()));
         self.error_message = Some(error);
         self.mode = AppMode::Error;
     }
     ```

Note: Error block uses "!" symbol styled with Meaning::Alert (red/warning color from theme). This matches the pattern used for ">" (Guidance) and "$" (Important).
